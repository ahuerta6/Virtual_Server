# Intro to Proxmox

[Proxmox](https://www.proxmox.com/en/downloads) is an open source server soloution that provides dowloads for iso files such as the Proxmox VE, Proxmox Backup Server, Proxmox Mail Gateway, and Proxmox Datacenter Manager.

In order to install it on a dedicated HDD or SSD you must first upload it a bootable USB drive with a software such as [Rufus](https://rufus.ie/en/). The better alternative in my opinion is using [Ventoy](https://www.ventoy.net/en/index.html). Ventoy is an opensource software that allows you to have multiple bootable install files all in one USB drive.

For the sake of showing you how proxmox looks when launching the install I used [Oracle VirtualBox](https://www.virtualbox.org/) to create a VM with Proxmox iso file. As you can see in Example 1.1 I created a VM in VirtualBox with 300gb of storage that proxmox is installing on. My real proxmox is on an entire hard drive dedicated to proxmox only.
