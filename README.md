# Proxmox Homelab Documentation
In this github repository I will show you my progress of setting up my own virtualized server. Eventually I will use this setup to run my own labs and tests. My goal is to simulat a network infrastructure. I am using Proxmox an open-source virtualization platform. Feel free to use this repo as a guide for your own setups. 

## Purpose

This Proxmox server is used to host multiple virtual machines for:

- Cybersecurity labs
- Network infrastructure
- Malware analysis
- Windows testing
- Linux administration
- macOS virtualization
- Penetration testing
